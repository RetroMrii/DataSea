# DataSea

DataSea is a full-stack analytics dashboard that helps authenticated users upload structured data files and understand them through charts, statistics, visual summaries, and concise textual insights.

## Tech Stack

### Frontend
- React + Vite
- JavaScript
- Tailwind CSS
- React Router
- Axios
- Context API
- Redux Toolkit

### Backend
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- JWT
- bcrypt
- Multer
- JOI
- Helmet
- express-rate-limit
